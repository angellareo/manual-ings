# Metodología incremental

El sistema se desarrolla y entrega en partes funcionales, cada una agregando más características.

## Ventajas
- Se obtiene software funcional desde las primeras entregas.
- Menos riesgo que en cascada.
- Facilita la detección temprana de problemas.

## Desventajas
- Planificación más compleja.
- Puede haber problemas de integración entre incrementos.
- No siempre se identifican bien los requisitos desde el inicio.
