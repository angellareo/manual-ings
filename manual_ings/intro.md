# Ingeniería del Software

Este es un proyecto vivo y **en construcción**.

Se trata de un libro ejecutable para utilizar como manual en una asignatura de [Ingeniería del Software](https://es.wikipedia.org/wiki/Ingenier%C3%ADa_de_software) para estudiantes de grado (por ejemplo, grado en Ingeniería Informática).

Si te interesa contribuir [puedes pasarte por el repositorio](https://github.com/angellareo/manual-ings) y consultar la [guía de contribuciones](https://github.com/angellareo/manual-ings/CONTRIBUTING.md).
