# Contribuir

¡Las contribuciones son bienvenidas y muy apreciadas! Cada pequeña ayuda cuenta, y siempre se dará el debido crédito. Puedes contribuir de las siguientes maneras.

## Informar Errores

Informa errores utilizando los issues de GitHub.

Si estás reportando un error, por favor incluye:

* El nombre y la versión de tu sistema operativo.
* Cualquier detalle sobre tu configuración local que pueda ayudar en la resolución del problema.
* Pasos detallados para reproducir el error.

## Corregir Errores

Revisa los issues de GitHub en busca de errores. Cualquier issue etiquetado con "bug" y "help wanted" está abierto para quien quiera solucionarlo.

## Implementar Funcionalidades

Revisa los issues de GitHub en busca de nuevas funcionalidades. Cualquier issue etiquetado con "enhancement" y "help wanted" está abierto para quien quiera implementarlo.

## Escribir Documentación

El proyecto Ingeniería del Software siempre puede beneficiarse de más documentación, ya sea como parte de la documentación oficial, en docstrings, o incluso en publicaciones de blog, artículos y similares.

## Enviar Comentarios

La mejor manera de enviar comentarios es creando un issue en GitHub.

Si estás proponiendo una nueva funcionalidad:

* Explica en detalle cómo debería funcionar.
* Mantén el alcance lo más reducido posible para facilitar su implementación.
* Recuerda que este es un proyecto impulsado por voluntarios, y que las contribuciones son bienvenidas. :)

## Primeros Pasos

¿Listo para contribuir? Aquí tienes cómo configurar `Ingeniería del Software` para el desarrollo local.

1. Haz un fork del repositorio en GitHub.
2. Clona tu fork en tu equipo.
3. Instala tu copia local dentro de un entorno virtual, por ejemplo, usando `conda`.
4. Crea una rama para el desarrollo local y realiza cambios en ella.
5. Haz commit de tus cambios y sube tu rama a GitHub.
6. Envía una pull request a través del sitio web de GitHub.

## Código de Conducta

Ten en cuenta que el proyecto Ingeniería del Software se publica con un [Código de Conducta para Contribuyentes](CONDUCT.md). Al contribuir a este proyecto, aceptas cumplir con sus términos.
